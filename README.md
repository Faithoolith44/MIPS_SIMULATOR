This project involves the design and implementation of a simplified MIPS simulator that focuses on the execution of R-type and immediate instructions. The simulator will provide a user-friendly interface through which users can input MIPS assembly code via a dedicated text field.

The system will include a visual panel that displays the current values of all 32 MIPS registers, allowing users to observe changes in real time as the program executes. Instruction execution will follow a step-by-step approach, where each instruction is processed sequentially. Users will control the execution flow using a “Next” button, which triggers the execution of one instruction at a time.

In addition, the simulator will display the corresponding machine code for each instruction in a separate section of the panel. This feature will enable users to understand the relationship between assembly instructions and their binary representations.
